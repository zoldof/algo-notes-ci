# 哲学
<details>
<summary>哲学の歴史と起源</summary>
◆歴史の流れ
・物理的存在段階：エネルギーと粒子の相互作用のみ
・生物的自己組織化段階：細胞が誕生し内外の境界が出来上がる
・感覚的統合：様々な刺激に対して統合的処理を行う神経系の発達過程
・知覚的世界観：世界と同化して反応する
・儀礼的行動：様々な出来事に意味付けをする
・精霊信仰：様々な現象の原因に精霊を関連付ける
・精霊交信：精霊信仰に儀礼的行動を付与したもの
・神話信仰：精霊と上位の神々で物語を構成する
・宗教哲学：宗教の存在意義や本質を究明する哲学
・理性哲学：［存在，理性，知識，価値，意識，言語］など総合的な問題について研究すること
↑既存宗教や神話を取り入れる形で進化してきた

◆哲学の起源
・ギリシアから始まったとされる
・・都市国家を発展させるための議論必要性
・・パピルスや羊皮紙の普及による影響
</details>
